Интранет: внутренняя частная сеть организации.

Строится на стеке протоколов TCP/IP

Реализуется технологиями WWW:

  • Язык гипертекстовой разметки документов HTML
  • Система доменных имен DNS
  • Протокол обмена гипертекстовой информацией HTTP

ИС, построенная по технологии Intranet, имеет структуру гипертекста, т.е. набора страниц, связанных между собой гиперсвязями. Это позволяет не синхронизировать информацию, тем самым "очищать" информационное пространство от "дубликатов"
